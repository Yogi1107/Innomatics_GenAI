# 📚 FastAPI Library Book System

## 🚀 Project Overview
A backend system to manage books, borrowing, and returns using FastAPI.

## 🔧 Features
- GET APIs
- POST with Pydantic validation
- CRUD operations
- Borrow/Return workflow
- Search, Sort, Pagination

## ▶️ Run Project
uvicorn main:app --reload

Open:
http://127.0.0.1:8000/docs
